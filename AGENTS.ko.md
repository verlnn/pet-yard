# Codex 프로젝트 메모

## 기본 응답 방식
- 기본적으로 한국어로 응답한다.

## 커밋 규칙
- 사용자가 구현 작업을 요청했고 별도로 커밋하지 말라고 하지 않았다면 작업 완료 후 자동으로 커밋한다.
- 커밋 메시지는 한글로 작성한다.
- 사용자가 한 번에 서로 다른 기능이나 변경사항을 여러 개 요청했다면, 가능하면 기능 단위로 나눠서 각각 별도 커밋한다.

## 프론트엔드 관련 문서
- 프론트엔드 전용 작업 규칙은 `web/AGENTS.md`에 둔다.
- 프론트엔드 전용 한글 문서는 `web/AGENTS.ko.md`에 둔다.
