## 계정별 접근 가능 기능

티어가 높아질수록 하위 티어의 권한을 자동으로 포함한다.

- Tier 0
    - 피드 열람
    - 지식 열람

- Tier 1
    - 피드 작성
    - 산책 게시글 열람
    - 산책 게시글 작성
    - 공개 모집 글 열람

- Tier 2
    - 산책 신청 / 매칭 / 채팅
    - 위탁 기능 이용
